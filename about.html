# About Me

Hi, I'm Julian, a young developer in the second half of his undergraduate degree at the University of Guelph.

I've been using computers my whole life but only discovered programming once I got to highschool, but once I found it, I never looked back. My experience is with C and Java but I've been teaching myself web development technologies in my spare time for the past year. I also love video games and the outdoors!

About this site

This site is part of my ongoing effort to teach myself web development. No backend work was done on this site; its all hosted using the amazing Github Pages. It was developed using the Harp static server and Jade and was built using Twitter's Bootstrap with a focus on minimalism.
