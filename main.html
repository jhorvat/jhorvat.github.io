Dialogc
2013

Dialogc was the assignment I wrote for CIS 2750. I have not had a chance to post the code on Github yet. The assignment was split into four parts each due over the course of the semester. The finished product was this:

  • 2 config file parsers in C; one written by hand and one created using flex and yacc. The goal of this parsing was to generate a GUI form written in Swing. The hand written parser would feed information through the JNI into a Java program which would generate the forms, the flex/yacc implentation was a command line application that generated the same form.
  • An IDE written in Swing for the config files complete with project directories and output directories for the generated forms. The IDE was capable of using either config file parser through a setting.
  • The generated forms would connect to a MySQL database and manipulate a table in said database.
  • A self-extracting install script written in Bash that would deploy the program and configure any settings in a dotfile in the user's home directory.
